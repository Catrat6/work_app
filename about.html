What is this ?

This is a simple web app built with Bootstrap and Google Forms made with the goal of bridging the communication gap between housekeeping and the rest of the store.

If you have any suggestions to improve it or would like to see something else here than please make sure to reach out.

How to use this app

  • Each department has a scannable QR code by the register.
  • When you need something cleaned in your area scan the code and click the button on the home page.
  • After clicking the button a form will load, fill out the form with as much info as possible and submit it.
